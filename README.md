# Epic to do app!
## Features:
- Add tasks and mark as complete
- View past completed tasks and delete or restore them
- Mark tasks as important to make them show up at the top of the todo list
- Assign tasks to different custom categories (with custom colors!)
- Search and filter tasks by categories
- Assign due dates to tasks and sort the todo list by due date or creation date
- Lots of gifs :3
